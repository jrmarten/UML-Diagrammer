Program Directory

The settings file in windows is found in ${APPDATA}\UML-Diagrammer

The settings file in other operating systems can be found in ${HOME}/.config/UML-Diagrammer/

Significant Files

Settings

Settings File Title

the primary settings file is called preferences.config

Notes

  • This file contains a set of key:value; pairs
  • Comments are the same as unix config comments, a line comment that starts with #

Settings values

Setting Variable Default Value Description
templateDir ${progDir}/Templates/ Where this program stores User-made templates
StyleSheet Which file to use as a style sheet
Collection Collection Which collection class should be used by the Source Code Generator for aggregation

Styles

Notes

  • The default name for this is nothing defining it in preferences makes the program look for a file.
  • This file should look like a regular CSS file
  • Class and id will not work in this program
  • Selectors do not have to be seperated by a comma
  • In addition to block comments ( /* ... */ ). Line comments are availible
  • Colors can be specified by name like Green, red, BLACK
  • Colors can be defined by an Integer such as 0x00FF00, #0000FF
  • Currently all Properties recongized are colors

Selectors

  • Class
    Property Default Value Description
    forground-color Black Sets Text color for Classes
    background-color White Sets background for Classes
    border-color Black Sets the color for the box Classes are in
  • Abstract Class
    Property Default Value Description
    forground-color Class.forground-color Sets Text color for Abstract Classes
    background-color Class.background-color Sets background for Abstract Classes
    border-color Class.border-color Sets the color for the box Abstract Classes are in
  • Iterface
    Property Default Value Description
    forground-color Class.forground-color Sets Text color for Iterfaces
    background-color Class.background-color Sets background for Iterfaces
    border-color Class.border-color Sets the color for the box Iterfaces are in
  • Inheritence
    Property Default Vaule Description
    forground-color Black Sets all color on Inheritence
  • Connection
    Property Default Vaule Description
    forground-color Black Sets all color on Connections
